# im.js.server
>[im.js](https://github.com/im-js/im.js) 服务端代码，基于 [cloverx](https://github.com/clover-x/cloverx) 开发。


## Requirements
* Node >= 7.6.0
* Redis >= 3.0.2
* Mysql >= 5.6.5

## Usage
先前往 `config` 目录，配置基本信息，然后  
```shell
npm install
```
启动
```shell
npm run dev
```
