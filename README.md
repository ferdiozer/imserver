# im.js.server
>[im.js](https://github.com/im-js/im.js) Sunucu kodu, göre [cloverx](https://github.com/clover-x/cloverx) geliştirildi.


## Requirements
* Node >= 7.6.0
* Redis >= 3.0.2
* Mysql >= 5.6.5

## Kullanımı
Önce "config" dizinine gidin, temel bilgileri yapılandırın, ardından
```shell
npm install
```
başlangıç
```shell
npm run dev
```
